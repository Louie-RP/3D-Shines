# 3D Shines

Dark, minimal storefront-style UI for a 3D printing shop.

## Pages

- Home: `index.html`
- Products: `products.html`
- About: `about.html`
- FAQ: `faq.html`
- Contact: `contact.html`
- Privacy: `privacy.html`
